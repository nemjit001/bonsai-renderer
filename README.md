# Bonsai Renderer

High performance render engine written in C++17.

## External dependencies

All of Bonsai's dependencies are included as git submodules in the repository and pinned at a specific
version.

- GoogleTest (v1.17.0)
- SDL3 (v3.2.x)
- SPDLog (v1.15.3)
- Volk (v1.4.321)
